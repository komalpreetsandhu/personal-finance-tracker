<h2 style="font-size:40px; font-weight:bold;">📌 Project Overview</h2>

<p style="font-size:20px; font-weight:bold;">
The <strong>Personal Finance Tracker</strong> is a Python-based project that helps users analyze <strong>spending habits</strong>. It processes transaction data, identifies trends, and provides actionable insights to improve budgeting and increase savings.
</p>

<p style="font-size:20px; font-weight:bold;">
The goal is to empower users to <strong>understand their financial behavior</strong> and make informed decisions.
</p>

---

<h2 style="font-size:40px; font-weight:bold;">🛠 Tools & Technologies</h2>

<ul style="font-size:20px; font-weight:bold;">
  <li><strong>Python</strong> – Core programming language</li>
  <li><strong>pandas & numpy</strong> – Data manipulation & analysis</li>
  <li><strong>matplotlib & seaborn</strong> – Data visualization</li>
</ul>

---

<h2 style="font-size:40px; font-weight:bold;">📝 Project Steps</h2>

<h3 style="font-size:28px; font-weight:bold;">1️⃣ Data Loading</h3>
<p style="font-size:20px; font-weight:bold;">Imported transaction data into Python using <strong>pandas</strong>.</p>

<h3 style="font-size:28px; font-weight:bold;">2️⃣ Data Cleaning</h3>
<ul style="font-size:20px; font-weight:bold;">
  <li>Converted <strong>Date</strong> column to <code>datetime</code> format</li>
  <li>Converted <strong>Amount</strong> to numeric type</li>
  <li>Removed duplicate transactions</li>
  <li>Filled missing <strong>Category</strong> values with "Other"</li>
</ul>

<h3 style="font-size:28px; font-weight:bold;">3️⃣ Data Transformation</h3>
<p style="font-size:20px; font-weight:bold;">Created a <strong>Month</strong> column to analyze spending trends over time.</p>

<h3 style="font-size:28px; font-weight:bold;">4️⃣ Pivot Table Analysis</h3>
<p style="font-size:20px; font-weight:bold;">Summarized spending by <strong>Category</strong> and <strong>Month</strong> using pivot tables.</p>

<h3 style="font-size:28px; font-weight:bold;">5️⃣ Data Visualization</h3>
<p style="font-size:20px; font-weight:bold;">Visualized spending patterns using:</p>
<ul style="font-size:20px; font-weight:bold;">
  <li>Monthly Spending by Category</li>
  <li>Category Distribution</li>
  <li>Spending Trends Over Time</li>
</ul>

<h3 style="font-size:28px; font-weight:bold;">6️⃣ Insights & Findings</h3>
<ul style="font-size:20px; font-weight:bold;">
  <li><strong>Utilities</strong> and <strong>Education</strong> accounted for the highest expenses</li>
  <li><strong>Dining Out</strong> and <strong>Entertainment</strong> had frequent smaller transactions</li>
  <li>Spending patterns varied across months</li>
</ul>

<h3 style="font-size:28px; font-weight:bold;">7️⃣ Savings Recommendations</h3>
<ul style="font-size:20px; font-weight:bold;">
  <li>Reduce <strong>Dining Out</strong> expenses</li>
  <li>Review <strong>Subscription Services</strong></li>
  <li>Set <strong>Category-based monthly budgets</strong></li>
</ul>
